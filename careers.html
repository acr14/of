Job Opportunities

Mobile Application Developer – iOS and Android


Description

The Mobile Application Developer role will design and develop mobile application as a member of our team. With experience in iOS, Android, and React Native development, this person will collaborate with members of the team to produce high quality UX solutions for our front-end mobile platform.

Responsibilities

  • Be a key contributor to all phases of software development mobile platform including ideation, prototyping, design, develop, and unit test
  • Advocate for clean design, maintainability and reusability
  • Build products, tools and services from scratch, as well as evolve existing systems
  • Work directly with the founders to set the technology and architectural roadmap
  • Ensure product quality and timeliness of work, provide advice and guidance, and resolve problems to meet objectives
  • Help develop new and refine existing processes to enhance quality and productivity
  • Play an active role in hiring and mentoring other engineers
  • Research and recommend tools and technologies in support of mobile development
  • Become a domain expert in real estate

Requirements

  • BS or MS in Computer Science, Information Systems/Technology, or equivalent discipline.
  • 2+ years programming native mobile applications specifically in iOS, Android, or React Native.
  • Familiar with UI/UX best practices and Google’s Material Design specification is a plus.
  • Experience publishing at least one consumer application to Apple App store and/or Google Play store.
  • Strong working knowledge of node.js, javascript, and JSX. Experience with Objective-C and Swift is a plus
  • Experience with OOP, MVC, and software design patterns.
  • Experience with data exchange (JSON, XML) and states (Redux)
  • Familiar with interacting with web services (REST, SOAP) and various formats (JSON, XML)
  • Familiar with developing real-time client interactivity using web sockets
  • Experience with production software development using tools such as Git, Jenkins, etc.
  • Familiar with creating and automating tests for mobile platforms
  • Familiar with customizing push notifications interactions
  • Integration with mobile analytic SDK (i.e. Google Analytics, Firebase, etc.)
  • Experience creating and maintaining technical design and systems architecture documentation.
  • Experience in Agile methodologies and developing working prototypes.
  • Must be able to work effectively and efficiently in a collaborative environment with founders, project managers, designers, and developers.

Software Development Engineer - Platform, Infrastructure, and Backend


Description

The Software Development Engineer role will design and build core backend services and features for performance, security, and scalability of the OfferFast platform. This person will collaborate with our front-end mobile developers to build tools, interfaces, and processes for efficient use of data structures and algorithms for performant services.

Responsibilities

  • Work with founders and team to design and implement new features for the OfferFast Application
  • Dedicated to optimizing the performance and scalability of the OfferFast infrastructure as the service continues to grow
  • Improve the reliability and efficiency of OfferFast's core services
  • Develop elegant solutions to complex problems
  • Code in Java, Javascript, and/or Python in a unix environment
  • Collaborate with front-end developers (iOS, Android, Web), on building new features, and iterating on existing features

Requirements

  • BS or MS in Computer Science, Information Systems/Technology, or equivalent discipline.
  • 2+ years professional software engineering experience
  • Strong background with algorithms and data structures
  • Demonstrate strong coding skills using Java, Python, C/C++, Golang, Scala, PHP (or equivalent programming language)
  • Experience with software engineering best practices (e.g. unit testing, code reviews, design documentation)
  • Extensive experience with web applications, backend infrastructure, distributed systems, performance analysis, and traffic manipulation
  • Be comfortable designing services on top of a cloud infrastructure e.g. AWS, Google Cloud Platform, Azure
  • Experience designing and implementing performant and fault-tolerant distributed systems
  • Proficient with both Relational and NoSQL databases (e.g. MySQL, Postgres, MongoDB)
  • Proficient with Google and Apple push notification services
  • Experience with source control management best practices to maintain Continuous Integration / Continuous Delivery
  • Experience with developing monitoring solutions to list areas of improvement through capture and comparison of metrics on uptime, performance, invalid requests, etc.
  • Experience creating and maintaining technical design and systems architecture documentation.
  • Experience in Agile methodologies and developing working prototypes.
  • Must be able to work effectively and efficiently in a collaborative environment with founders, project managers, designers, and developers.
